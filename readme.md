# LINE Bot - 智能助理

這是一個使用 Python 開發的 LINE Bot 智能助理，提供了多種實用功能，包括：

- **縮網址**：將長網址轉換為簡短的網址
- **匯率轉換**：查詢即時的匯率資訊
- **股票資訊**：查詢台灣股市相關資訊，包括個股基本面、交易統計等
- **天氣查詢**：查詢 36 小時內的天氣預報和即時觀測資訊
- **新聞搜索**：搜索相關新聞文章並顯示摘要資訊
- **待辦事項管理**：提供待辦事項的新增、刪除、更新和查詢功能


## 使用說明

1. 用戶需要先在 LINE 上加為好友。
2. 開始聊天並輸入上述各種指令，即可獲得相應的功能服務。


此外，還使用了以下公開 API 資料源：
- 天氣 API
- 匯率 API
- 股票資訊 API
- 縮網址 API
- 股票 API
